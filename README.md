# Image-Board
# what is Image-Board?
it's a Single page application where you can upload and browse images.
# How can i run it ?
clone it then - npm start -
+ Runs the app in the development mode. Open http://localhost:8080 to view it in the browser.
# Technologies:
* Vue.js
* Node.js
* Express
* PostgreSQL
* AWS S3
# Features:
* Upload a photo to AWS S3 from a file.
* you can comment on images and see other's comments
* more button to see olders photos
